Zurfur is a decentralized art commission platform built on AT Protocol (Bluesky). Core vision: "Super App" for furry creator economy — consolidates commissions, invoicing, chat, profiles with data sovereignty via AT Protocol.

**Current state (2026-04-07):** Feature 1.1 (AT Protocol OAuth) is complete. The critical path is: Auth → Identity/Profile (Feature 2) → Artist TOS (Feature 10) → Commission Engine (Feature 3) → Financial Gateway (Feature 4) → Notifications (Feature 9).

**Why:** The platform is designed to solve trust, data ownership, and workflow fragmentation problems in the furry art commission space. AT Protocol integration ensures users own their data and reputation.

**How to apply:** All work should advance along the critical path. Feature 2 (Identity & Profile) is the immediate next priority. Design docs in design/ are authoritative for feature specs.
